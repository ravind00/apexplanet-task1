# My First Web Development Project 🚀

Welcome to my very first web development project! This webpage was created as part of my first internship as a Web Developer at **ApexPlanet Software Pvt. Ltd.**

## 📝 About the Project

This is a simple, responsive, and interactive "About Me" webpage. It demonstrates the fundamental concepts of web development by combining structure, styling, and behavior in a single web document.

## ✨ Features

* **Clean UI:** A centered card layout with a light and modern color scheme.
* **Typography & Colors:** Customized fonts and text colors for headings and paragraphs.
* **Interactivity:** A functional button built with JavaScript that triggers an alert message when clicked.
* **Social Links:** Direct links to my LinkedIn and GitHub profiles.

## 🛠️ Technologies Used

* **HTML5:** For the structure and content of the webpage.
* **CSS3:** For styling, layout (Flexbox), and visual aesthetics (shadows, border-radius, hover effects).
* **Vanilla JavaScript:** For adding interactivity to the button.

## 🚀 How to Run

Since this project uses basic web technologies, no installation or setup is required. 

1. Clone or download this repository to your local machine.
2. Ensure that the image file (`images.jpg`) is in the same folder as the HTML file.
3. Double-click the `task1.html` file to open it in any modern web browser (Chrome, Firefox, Edge, etc.).
